# vite-vue2-simple-starter

> ### Vue2 + Vite + Vuex + Vue-router + Axios

We provide sample Code about Vuex and Vue-router and Axios

> #### If you want to use this starter

```
npm install
```

> #### If you want run this starter

```
npm run serve
```

> #### If you want build starter

```
npm run build:prod
```
